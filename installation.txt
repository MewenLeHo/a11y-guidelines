*** Build du projet ***
La commande "grunt" a la racine du projet permet de compiler le projet, les pages web sont crees dans le repertoire "output".
Pour creer une version destinee a etre mise en production utiliser la commande : "grunt prod". Cela permet l'ajout de l'identifiant pour google analytics dans le code (fichier /config/config-prod.js)
